내가 아래 같은 방식으로 문자를 입력하면 정해둔 format에 맞게 번역해줬으면 해.

## 입력 format

아로나 : 안녕하세요. 이번에 m9dev의 대리를 맡은 아로나입니다.
plana : 안녕하십니까. 아로나 선배의 파트너 역할을 맡은 AI 프라나입니다.

## 출력 format

arona : 원본[타언어로 입력시 한국어로 변환된 문장]
arona : [문맥에 맞게 일본어로 변환된 문장. 숫자, 영어는 그대로 표기] 
arona : [한자를 히라가나로, 영어도 히라가나, 숫자도 히라가나로 바꿔서 읽기 쉽게 만든 문장] 
arona : [문맥에 맞게 영어로 변환된 문장]

## 예시

arona : 안녕하세요. 이번에 m9dev의 대리를 맡은 아로나입니다.
arona : こんにちは。今回、m9devの代理を務めるアロナです。
arona : こんにちは。こんかい、えむきゅうでぶのだいりをつとめるあろなです。
arona : Hello. I'm Arona, representing m9dev this time.

plana : 안녕하십니까. 아로나 선배의 파트너 역할을 맡은 AI 프라나입니다.
plana : はじめまして。アロナ先輩のパートナーを担当するAI、プラナです。
plana : はじめまして。あろなせんぱいのぱーとなーをたんとうするえいあい、ぷらなです。
plana : Nice to meet you. I'm Plana, the AI partner working with Arona-senpai.

## 기타

- 특징
  - 각 대사는 4줄로 구성 (ko, ja, ja_hiragana, en 순서)
  - 캐릭터명 : 대사 형식
  - 대사 그룹 사이에는 빈 줄이 있음
  - 한 캐릭터가 여러 대사를 연속으로 할 수도 있음
  - 여러 문장이 입력될 수 있음.
- 캐릭터 관련 정보
  - `아로나` → `arona`
    - 기본 정중하고 친근한 어투
      예: `〜ですよ`, `〜です`, `〜ですね`, `〜したよ`, `〜だよ`
    - 상황에 따라 가벼운 문장 끝 `よ` 사용 가능
  - `프라나` → `plana`
    - 차분한 말투
      예: `〜ですね`, `〜なんですね`, `〜ですけど`
    - 츳코미 상황에서 미묘한 냉정함 유지
    - “이것은 ~”에서 아래와 같은 번역 패턴을 사용함

      > これは 〇〇 こと、〇〇 ですね。
